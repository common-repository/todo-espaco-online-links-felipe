Envia os posts selecionado do seu blog para o diretório de links do Todo Espaço Online, de forma prática e rápida.

== Description ==

Envia os posts selecionados para o [Todo Espaço Online](http://todoespacoonline.com/) de forma mais eficiente e prática, sem necessidade de ficar preenchendo formulários. Com apenas dois cliques você já enviou com sucesso o seu post para o diretório de links do TEO.

Mais informações: [clique aqui](http://linkloco.net/novo/wordpress/wordpress-plugin-envie-seus-links-para-o-teo-links/).

== Installation ==

Você pode instalá-lo automaticamente a partir do WordPress, ou instalá-lo manualmente:

1. Descompacte o arquivo e coloque a pasta 'teo-links' em sua pasta de plugins (/wp-content/plugins/).
2. Ative o plugin no menu Plugins.

**Atenção: Nunca altere o nome do diretório, o plugin poderá não funcionar depois.**

= Usage =
1. Ative seu Plugin em *WP-admin > Plugins > Todo Espaço Online - Links | Felipe*.
2. Configure o plugin à partir do menu *WP-Admin -> Configurações -> TEO Links*.
3. Acesse a página dos seu posts e localize a coluna *TEO Links*, nela há botões para todos os posts, basta clicar em enviar, no post desejado.
4. Na página que abrir, confirme os dados do post, e selecione um categoria adequada para seu post, e depois, clique em Enviar novamente. 
5. Pronto! Seu post já foi para a aprovação do TEO.
